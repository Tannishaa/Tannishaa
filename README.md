### Cloud & Backend Projects (The "Winter Arc" Series)
*These are projects I built to bridge the gap between my AWS certification theory and practical, hands-on application.*

**1. Flux-Order: High-Concurrency Ticketing Engine**
> *An exploration into handling flash sales without inventory overselling.*
* **Tech:** AWS (SQS, DynamoDB), Terraform, Redis, Streamlit.
* **What I learned:** Implemented Redis locking (SET NX) to prevent race conditions and used Terraform to automate the deployment of my AWS resources.

**2. Bill-E: Serverless Audit & Risk Engine**
> *An event-driven pipeline designed to flag anomalous financial transactions.*
* **Tech:** Python, AWS (S3, SNS, Lambda).
* **What I learned:** Built a decoupled, trigger-based architecture where uploading a file to S3 triggers a Python script to evaluate risk, immediately alerting via SNS if flagged. 

**3. Vibe-Check: Real-Time Sentiment Analysis API**
> *A serverless API for quick customer feedback analysis.*
* **Tech:** AWS Lambda, API Gateway, SNS.
* **What I learned:** Designed a cost-effective, rule-based filtering system to process text inputs and send automated email alerts via SNS for negative feedback.
