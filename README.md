### Proof of Work & Cloud Architecture
*Projects built to bridge the gap between architectural theory and production-grade implementation.*

**1. Bill-E: Serverless Audit & Risk Engine** | [Live Demo](https://bill-e-audit.vercel.app/) • [GitHub](https://github.com/Tannishaa/bill-e-audit)
> *An event-driven pipeline designed for 100% operational cost-efficiency.*
* **Tech:** Python, AWS (Lambda, SQS, S3, DynamoDB), Terraform, Next.js.
* **Architecture:** Architected a zero-idle-cost ingestion pipeline that decouples storage from OCR processing. Engineered fault-tolerant Lambda execution with graceful degradation for API timeouts and automated SNS alerts for high-risk anomalies.

**2. Flux-Order: High-Concurrency Ticketing Engine** | [Live Demo](https://flux-cinema-frontend.vercel.app/) • [GitHub](https://github.com/Tannishaa/flux-order)
> *An exploration into handling flash sales and preventing double-booking.*
* **Tech:** Next.js, Python, Redis, AWS (SQS, DynamoDB).
* **Architecture:** Implemented Redis Distributed Locks (Mutex) to resolve critical race conditions and engineered an idempotent background worker service to process asynchronous orders without duplicate transactions.

**3. Sentinel: AI-Driven Network Intrusion Detection**
> *A real-time active network security tool.*
* **Tech:** Python, FastAPI, Scapy, Machine Learning.
* **Architecture:** Engineered a real-time NIDS backend to analyze high-throughput network packets, flagging malicious traffic for downstream ML models, and serving threat intelligence via a low-latency REST API.
