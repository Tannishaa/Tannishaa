### Proof of Work & Cloud Architecture
*Projects built to bridge the gap between architectural theory and production-grade implementation.*

**1. Sentinel: AI-Driven Network Intrusion Detection System**
> *A real-time active network security tool.*
* **Tech:** Python, Scapy, FastAPI, Machine Learning.
* **Architecture:** Engineering the backend to sniff and parse high-throughput network packets in real-time, feeding data into anomaly detection models and serving threat intelligence via a low-latency REST API.

**2. Flux-Order: High-Concurrency Ticketing Engine**
> *An exploration into handling flash sales and preventing double-booking.*
* **Tech:** Next.js, Python, Redis, AWS (SQS, DynamoDB).
* **Architecture:** Implemented Redis Distributed Locks (Mutex) to resolve critical race conditions and engineered an idempotent background worker service to process asynchronous orders without duplicate transactions.

**3. Bill-E: Serverless Audit & Risk Engine**
> *An event-driven pipeline designed for 100% operational cost-efficiency.*
* **Tech:** Python, AWS (S3, Lambda, SNS), OCR.
* **Architecture:** Built a zero-idle architecture where uploading a file to S3 triggers a script to evaluate fraud risk, automating immediate SNS alerts to auditors for high-risk anomalies.
